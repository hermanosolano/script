Criado em 2008 e comprado pela Microsoft em 2018, o Github é uma plataforma de hospedagem e gerenciamento de códigos que permite a interação entre os usuários e a colaboração com esses projetos, contribuindo assim para o crescimento da comunidade open source. Ele também é considerado como uma rede social onde a gente pode criar nossas próprias coisas, seguir projetos/pessoas, comentar projetos, receber/fazer contribuições em projetos, isto é, é o local onde podemos ter acesso a comunidade de desenvolvimento do mundo todo.

Então, basicamente é um repositório online, onde a gente pode armazenar os nossos códigos, e com isso gerenciá-los a partir dali e interagir com outras pessoas e projetos de qualquer lugar do mundo.

Acesse https://github.com. Temos a página inicial do Github. É importante navegar pelo o Github antes mesmo de criar uma conta. Vemos aqui por exemplo que o Github é usado por empresas renomadas como Google, Facebook entre outras.
No lado superior direito temos o Menu.

Em WHY GITHUB temos algumas páginas explicando algumas coisas que fazem parte do Github.

Em ENTERPRISE é uma parte voltado para empresas.

Em EXPLORE é a forma geral de pesquisar no Github, seja por repositórios populares, tópicos, os Marketplaces mais usados.

Em MARKETPLACE contém ferramentas que adicionam funcionalidades e melhoria no fluxo de trabalho nos repositórios onde os usuários podem baixar e instalar, como se fosse plugins.

Em PRICING é a comparação e diferenças dos planos que o Github oferece.
